Проект "Сложно сосредоточиться"
В этом проекте я сверстала адаптивный сайт в светлой и тёмной темах, а также настроила автоматическое переключение 
между ними в зависимости от системных настроек пользователя. Написала разметку, перенесла контент из макета и всё это 
стилизовала при помощи CSS. 
 Технологии:
- HTML
- CSS
- Flexbox
- Grid Layout
- CSS-переменные
- Медиа-запросы
